## 🎓 ÖĞRENCİ

### Hızlı Rezervasyon
```
Öğretmen → Rezervasyon Yap → Tarih Seç → Slot Seç → Gönder → ✅
```

### Slot Yok?
```
→ Başka tarih dene
→ Veya süreyi düşür (60dk → 30dk)
```

## 🔥 EN ÇOK KULLANILAN

### Öğretmen

1. **Hafta İçi 9-17:** ⚡ → Option 1
2. **Pazartesi'yi Kopyala:** 📋 → Pazartesi → Tümü
3. **Yaz Tatili:** 🏖️ → Tatil Dönemi → 1-31 Ağustos
4. **Özel Gün:** 🏖️ → Özel Saatler → 20 Ocak 14-16

### Öğrenci

1. **Rezervasyon:** Tarih seç → Slot seç → Gönder
2. **Boş Slot Yok:** Başka tarih dene
3. **Daha Çok Seçenek:** Süreyi düşür (60→30)

## 📞 HIZLI DESTEK

**Öğretmen Sorunu:**
- Takvim ekleyemiyorum → Önceden aynı saat var mı kontrol et
- Tatil ekleyemiyorum → Geçmiş tarih mi? Gelecek tarih seç
- Slot silemiyorum → Long press dene

**Öğrenci Sorunu:**
- Slot göremiyorum → Öğretmen o gün izinli veya dolu
- Rezervasyon başarısız → Slot dolu olmuş, yeni slot seç

**Teknik Sorun:**
- Yüklenmiyor → İnternet kontrolü
- Hata veriyor → Çıkış yap, tekrar giriş yap
- Devam ediyorsa → support@terenceeegitim.com
